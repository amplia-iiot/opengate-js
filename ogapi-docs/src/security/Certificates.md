## create(rawFile)


This invoke a request to OpenGate North API and the callback is managed by promises
This method create an element certificate

### Parámetros

| Nombre | Tipo | Opcional | Descripción |
| :--- | :--- | :---: | :--- |
| **rawFile** | `File` | ❌ | this File is the certificate |

### Retorna

{{% notice tip %}}
**Tipo:** `Promise`
<br>

{{% /notice %}}
